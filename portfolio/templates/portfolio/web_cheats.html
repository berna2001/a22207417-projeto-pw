Experiencia na Criacao do Site

    Este projeto que me foi pedido no âmbito da disciplina de Programação Web permitiu-me descobrir e aprofundar o meu conhecimento âmbito das linguagens de Django, Python, HTML5, CSS, Bootstrap e JavaScript.

    Esta ligação entre o curso de Informática de Gestão e a necessidade do conhecimento na criação de sites, na minha opinião, faz todo o sentido e e com certeza uma área que tenciono explorar com mais profundidade.

Tecnologias Utilizadas

  • Django - Framework web em Python para desenvolvimento rápido e design limpo.
  • Python - Para a criação de modelos e classes.
  • HTML5 e CSS - Para a estrutura e o estilo das páginas web.
  • Bootstrap - Framework CSS para um design responsivo e componentes prontas para uso.
  • JavaScript - Para funcionalidades interativas e dinâmicas.
  • Git - Controlo de versão do código.
