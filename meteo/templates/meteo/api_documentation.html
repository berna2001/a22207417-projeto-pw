Documentação da API Meteo

Endpoints

Lista de Cidades

GET /meteo/api/cities/

Retorna a lista de cidades com previsões meteorológicas disponíveis.

Previsão Meteorológica de Hoje para uma Cidade

GET /meteo/api/weather/today/<city_id>/

Retorna a previsão meteorológica de hoje para a cidade especificada.

Parâmetros:

  • city_id (integer): O ID da cidade

Previsão Meteorológica para os Próximos 5 Dias para uma Cidade

GET /meteo/api/weather/five_days/<city_id>/

Retorna a previsão meteorológica para os próximos 5 dias para a cidade especificada.

Parâmetros:

  • city_id (integer): O ID da cidade
